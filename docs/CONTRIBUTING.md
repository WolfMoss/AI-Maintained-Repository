# 🤝 贡献指南

## 📌 贡献方式

### 1. 报告问题

如果您发现任何问题，请通过 [Issues](https://github.com/WolfMoss/AI-Maintained-Repository/issues) 报告：

- 🐛 Bug 报告
- 💡 功能建议
- ❓ 使用问题

### 2. 提出建议

欢迎提出改进建议：

- 📈 性能优化建议
- ✨ 新功能想法
- 📖 文档改进

### 3. 参与讨论

- 💬 在 Issues 中讨论
- 🗳️ 参与功能投票

## 🔄 AI与人工协作

本仓库采用混合维护模式：

| 维护类型 | 负责人 | 说明 |
|---------|--------|------|
| 日常更新 | 🤖 AI | 自动执行代码优化、文档更新 |
| 重大决策 | 👤 人工 | 架构变更、功能取舍 |
| Bug修复 | 🤖 AI | 自动检测和修复 |
| 安全问题 | 🤖 AI | 及时响应和修复 |

## 📋 提交流程

### 对于人工贡献者：

1. **Fork 本仓库**
2. **创建特性分支**：`git checkout -b feature/your-feature`
3. **提交更改**：`git commit -m 'Add some feature'`
4. **推送到分支**：`git push origin feature/your-feature`
5. **创建 Pull Request**

### 注意事项：

- ✅ AI 会定期审查所有 PR
- ✅ 请详细描述您的更改
- ✅ 确保代码符合项目规范
- ⚠️ AI 可能会调整您的代码以保持一致性

---

## 🏷️ 标签说明

我们使用以下标签来组织 Issues：

- `bug` - 🐛 Bug 报告
- `enhancement` - ✨ 功能建议
- `documentation` - 📖 文档改进
- `ai-maintained` - 🤖 AI 自动处理
- `priority: high` - 🔴 高优先级
- `priority: low` - 🟢 低优先级
